Panduan Instalasi API

Pra-kata Instalasi

  1. Install Go-lang
  2. Sesuaikan $GOPATH dengan preferensi Anda
  3. Clone source code dengan command
    go get -u https://github.com/2021-IT-Pemrograman-Integratif/project-2-mhilmi999.git

Depedensi

  1. Gunakan go module go mod init
  2. Download module yang digunakan go mod tidy
  3. Build project go build
  4. Dapat juga menggunakan depedency manager dengan mengetikan dep ensure

Konfirgurasi DB

  1. Buat baru dan import DB dari repo pada SQL Anda
  2. Edit file config.json pada folder config
  3. Konfigurasikan "DB_USERNAME", "DB_PASSWORD", "DB_PORT", "DB_HOST", "DB_NAME" sesuai preferensi
  4. Simpan konfigurasi database

Run Web Server

  1. Jalankan dengan command go run main.go
  2. Buka browser dan ketikan URL localhost:1234/
